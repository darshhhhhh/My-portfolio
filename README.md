# Darshil Vaja Portfolio Website

A seamless, modern, slightly dark 3D portfolio website for a Machine Learning Engineer profile.

## Latest design changes

- Removed the `Open to AI/ML Engineer roles` pill from the home section.
- Removed visible section divider lines.
- Made the background continuous across the full website.
- Added smoother reveal transitions.
- Added 3D depth and interactive tilt effects for cards, visuals, metrics and contact cards.
- Improved glassmorphism, shadows, lighting, and depth.
- Kept natural continuous scrolling without slide snapping or forced pauses.

## Structure

```text
portfolio-website/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── resume/
```

## How to run locally

Open `index.html` directly in your browser.

Recommended:

1. Open the folder in VS Code.
2. Open `index.html`.
3. Right-click and select `Open with Live Server`.

## How to add your resume

Replace this file:

```text
assets/resume/resume.pdf
```

with your actual resume PDF using the same filename.

## How to host on GitHub Pages

1. Create a GitHub repository.
2. Upload all files from this folder.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
